# myPointers

Реализация упрощённых аналогов `SharedPtr` и `WeakPtr` на C++20.

Проект демонстрирует модель разделяемого владения объектом через control block, работу слабых ссылок, базовую семантику копирования и перемещения.

## Возможности

- `SharedPtr<T>` с разделяемым владением
- `WeakPtr<T>` для невладеющего наблюдения за объектом
- подсчёт ссылок через control block
- поддержка копирования и перемещения для обоих указателей
- поддержка `WeakPtr::lock()`

## Устройство

### `SharedPtr<T>`

`SharedPtr` хранит указатель на объект и указатель на control block. Несколько экземпляров `SharedPtr` могут совместно владеть одним объектом. Объект уничтожается, когда счётчик сильных ссылок достигает нуля.

### `WeakPtr<T>`

`WeakPtr` не владеет объектом и не влияет на время его жизни. Он используется для безопасного наблюдения за объектом и получения временной сильной ссылки через `lock()`.

### Control block

Control block хранит счётчики сильных и слабых ссылок и отвечает за уничтожение управляемого объекта. Удаление выполняется через объект control block.
